#  Real-Time Weather Forecast Application (Python)

##  Overview
This is a **Real-Time Weather Forecast Application** developed using **Python**.  
The application fetches live weather data from a weather API and displays current weather details such as temperature, humidity and weather conditions for a given location.

This project helps in understanding **API integration, Python programming, and real-time data handling**.

---

##  Features
-  Get real-time weather details using city name  
-  Displays temperature in Celsius  
-  Shows humidity percentage   
-  Shows weather condition  
-  Fetches live data from API  

---

##  Technologies Used
- Python 3  
- Weather API (OpenWeatherMap)  
- Requests Library
